 Requirements:
 
Python 3.11+
Django
Django REST Framework

 cmds:
git clone 

cd ayahmood

pip install django djangorestframework requests

for db : python manage.py migrate

python manage.py runserver

http://127.0.0.1:8000/api/ayah/sad/
